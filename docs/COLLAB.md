# Collaboration Guidelines

Technical Team collaboration for this project is done via IRC hosted by Mozilla.  

```
Channel: #gcppoc
Server: irc.mozilla.org
Proxy: None
Chat SSL Server Port: 6697
```

**General Guidelines**
1) The IRC server is public, keep discussion limited to technical details of the project.
2) Discussion of anything related to contracts, pricing, general business, or other sensitve material should be done via Google Hangouts or other secure means.
3) Please do not put status reports in IRC, they should be communicated via email.
4) Issues or tasks need to be tracked via Github issues on the `mozilla-rust-sdk` project.
